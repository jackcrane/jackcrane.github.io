What is a static website?

A static website is the simplest and cheapest way to go about creating and maintaining a website.

Static websites consist of no backend code (code that is executed on a server that the user doesn't have access to)

Pros:

  • Extremely simple
  • Easy to create
  • Easy to mantain
  • Low operation and creation costs
  • Few security risks

Cons:

  • No online payments
  • No forums
  • Basic form handling

My base price for a static website is $50, with ~$10 monthly operational costs

What is a dynamic website

A dynamic website is the most professional way to create a website, but it will be much more expensive.

You should be looking into a dynamic website if you seek to create a unique experience for your customers, accept online payments, or create a login system or forum

Pros:

  • Futureproofing: with a dynamic start, it will be easier to grow
  • Expanded capabilities like forums, online ordering/payment, or account systems

Cons:

  • Will likely become complex very quickly
  • Much more expensive to create and operate
  • More security risks and vulnerabilities

My base price for a dynamic website is $500, with ~$50 monthly operational costs
